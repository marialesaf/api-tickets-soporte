# 🎫 RainForge - API de Tickets de Soporte

Este es el backend desarrollado en **NestJS** y **Prisma ORM** para la gestión interna de tickets de soporte técnico de la agencia.

---

## 🛠️ Tecnologías Utilizadas
* **Framework:** NestJS
* **ORM:** Prisma v6
* **Base de Datos:** SQLite (Local)
* **Validaciones:** Class-validator & Class-transformer
